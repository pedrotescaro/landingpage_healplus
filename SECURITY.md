# Security Policy

## Reporting a vulnerability

If you find a security issue in this repository, please report it privately.

Preferred contact:

- Email: healgrupo@gmail.com

Please include:

- a clear description of the issue
- reproduction steps
- affected files or sections
- possible impact
- screenshots or proof of concept when relevant

## Response

We will review reports as soon as possible and evaluate:

- severity
- scope
- remediation priority

Please do not publish security issues publicly before the project has had a
reasonable chance to investigate and address them.

## Scope

This policy applies to:

- repository source files
- public landing content
- client-side behavior implemented in this project

It does not automatically cover third-party hosting providers, browser
extensions, or external services outside this repository.
